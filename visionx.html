03. VisionX Website

Timeline

3 weeks

Platform

Wordpress Website

My Role

UI/UX Designer

Front End Developer

Introduction

VisionX is an optometry specialist in Parnell who recently went through a new store renovation and was looking for an updated website to reflect the updated style and the recent branding changes.

Defining the problem

To align the online presence of VisionX with the physical renovations of the main branch, we want to design and develop a new responsive website with a "modern and clean" aesthetic that provided an easy navigating experience.

In an interview with Richard, he revealed some critical pain points that he had previously experienced with websites, including responsiveness on different devices and slow load times of modern web pages.

Richard expressed that although the primary method of communication with his customers is through social media and in-person, he would like to be able to advertise new promotions on his website.

He also stated that the website had an existing web store that he would like integrated as an extension of this project at a future date.

Goals

After a preliminary interview with Richard and the staff of VisionX, the scope of the project, the reasons for building the new website and how it was going to be built were defined.

The main goals of this project were:

Update and modernise the existing website

Aside from aesthetic updates, an improvement to simplify navigation and reduce the number of clicks to get to vital information was needed.

Better communicate values and campaigns

In attempts to attract new customers to the store, an updated online presence that allows for the latest promotions and reflects the values of VisionX are needed.

Key requirements for this project included:

  • An updated "modern and clean" yet familiar look to the new website

  • Easy responsive navigation for multiple devices (mobile, tablet, desktop)

  • Fast loading times for content
